# Simple Offline Game Platform

## UI/UX
- Main Menu
    - Games Option:
        - Goes into game selection scene
    - Options/Setting:
        - Goes to options/setting scene
    - Exit:
        - Exits the game

- Game Selection Scene
    - Lists the **available games** to be played
    - The player must be able to select and play a game

- Options/Settings Scene
    - Credits:
        - Display group name and members
    - Instructions:
        - Displays how to use the program

- Game Proper
    - Pause/Menu Scene:
        - The games must have a pause/menu scene to display with several options
    - Restart Option:
        - Restarts the game
    - Main Menu Options:
        - Goes to the main menu
    - Exit Option:
        - Exits the game platform

## Functionality
- Note that the player must be able to easily navigate between the different parts of the game.
- In terms of the features of the game manager, the project must be able to do these actions:
    - Easily removing or adding a game
- For demonstration purposes, the program must have three games added to it:
    - Wordle
    - Blackjack
    - Hangman
